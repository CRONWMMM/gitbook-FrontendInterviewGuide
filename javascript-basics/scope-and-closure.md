# 作用域和闭包

> 简单来说，**作用域** 指程序中定义变量的区域，它决定了当前执行代码对变量的访问权限。

### 词法作用域

词法作用域（`Lexical Scopes`）是 JavaScript 中使用的作用域类型，词法作用域也可以被叫做静态作用域，与之相对的还有动态作用域。
